# Hospital Performance & Patient Experience Dashboard

## Tools Used
- Power BI

## Project Overview
This project analyzes hospital performance and patient experience data to evaluate healthcare quality across different regions and hospital types using an interactive Power BI dashboard.

## Key Insights
- Comparison of hospital performance by state, city, and hospital type  
- Analysis of patient experience scores and readmission rates  
- Evaluation of patient safety and care effectiveness metrics  
- Identification of regions with high readmission rates and lower patient satisfaction  

## Dashboard Features
- Interactive KPIs and visualizations  
- Filters for state, city, and hospital category  
- Clear and user-friendly dashboard design for quick analysis  

## Files Included
- Hospital_Performance.pbix – Power BI dashboard file  
- Hospital_Data.xlsx – Dataset used for analysis  

## Business Value
The dashboard supports data-driven decision-making by helping healthcare stakeholders identify performance gaps and improve patient care quality.
